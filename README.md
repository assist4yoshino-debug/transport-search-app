# ASSIST ROUTE FINDER

## 今回の検索仕様
1. 発地市場・着地市場を選択
2. まず「市場 → 市場」で検索
3. 該当がなければ、自動で「都道府県 → 都道府県」に切り替えて検索

## データ更新
- carriers.csv を編集します。
- 市場単位の対応が分かる会社は、以下2列に市場名を入れてください。
  - origin_markets
  - dest_markets

## 例
北部大果 → 荏原青果 の市場単位対応がある会社の場合：

origin_markets に `北部大果`
dest_markets に `荏原青果`

複数ある場合は `、` 区切りです。

## 注意
origin_markets / dest_markets が空の会社は、市場単位検索では該当しません。
市場で該当がない場合のみ、都道府県単位の検索結果に出ます。
